一个用 Python 写的待办事项管理工具，支持增删改查，数据通过 JSON 文件持久化存储。

## ✨ 功能

- ➕ 新增任务
- ✅ 标记完成
- 🗑️ 删除任务
- 📋 查看所有任务
- 💾 数据持久化（JSON 文件）

## 🚀 快速开始

### 依赖
```bash
无额外依赖（仅用标准库）
```

### 运行
```bash
python todo.py
```

## 📖 使用说明

运行后选择菜单数字操作，数据自动保存在 `todos.json` 文件中。

## 🛠️ 用到的知识点

| 知识点 | 应用 |
|--------|------|
| 列表和字典操作 | 存储任务数据 |
| JSON 文件读写 | 数据持久化 |
| 函数拆分 | 每个功能一个函数 |
| 异常处理 | 处理文件读写错误 |
| 日期时间处理 | 记录任务创建时间 |
